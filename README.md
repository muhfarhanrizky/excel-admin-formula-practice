# excel-admin-formula-practice
This project demonstrates how Excel formulas can be used to analyze employee data and answer common administrative questions.

The dataset contains employee information such as salary, bonuses, overtime pay, marital status, number of children, birth date, and employment date.

Using Excel formulas, this project calculates and analyzes:

- Employee ID
- Date of birth
- Employee age
- Employee status
- Employee and marital status
- Work tenure
- Salary and total income
- Overtime bonus and THR bonus
- Child allowance
- Total deductions and contributions
- Net income before and after deductions
- Cash denomination calculation

This project highlights practical Excel skills commonly used in administrative related tasks.
